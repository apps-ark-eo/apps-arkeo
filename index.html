🔒 Politique de confidentialité

Arkeo Apps — apps.ark-eo.com

Données collectées
Nos applications collectent uniquement les données nécessaires à leur fonctionnement : notes vocales (stockées localement sur votre iPhone), préférences d'utilisation (stockées en local via UserDefaults), et adresse email si vous nous contactez volontairement.

Clé API
Votre clé API Anthropic est stockée exclusivement dans le Keychain iOS chiffré de votre appareil. Elle n'est jamais transmise à nos serveurs.

Données tierces
MindFlow utilise l'API Anthropic pour la classification des notes. Les textes dictés sont transmis à Anthropic pour traitement selon leur propre politique de confidentialité. Aucune autre donnée n'est transmise à des tiers.

Vos droits (RGPD)
Vous pouvez à tout moment supprimer vos données depuis l'application. Pour toute demande : contact@ark-eo.com
